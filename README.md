# Axpen backend
---
## Step by step guide to set up 

### Clone repo
```bash
git clone https://github.com/m1a2023/axpen-back.git
```
---

### Enter the dir
```bash
cd axpen-back
```
---

### Install dependencies
```bash
npm install
```
---

### Run server
```bash
nodemon server.js
```
---

### Done
